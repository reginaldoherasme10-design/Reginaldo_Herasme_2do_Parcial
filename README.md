Sistema de Gestion de Calificaciones
Segundo Parcial - Programacion en C++
Que hace?
El programa permite registrar las calificaciones de hasta 20 estudiantes y muestra estadisticas del
grupo: promedio, nota mas alta, nota mas baja y quienes aprobaron.
Funciones del programa
• registrarDatos() — pide los nombres y notas de cada estudiante
• calcPromedio() — calcula el promedio del grupo
• notaAlta() — devuelve la nota mas alta
• notaBaja() — devuelve la nota mas baja
• mostrarDatos() — muestra el resumen general
• verAprobados() — lista a los estudiantes con nota >= 70
• menu() — muestra las opciones disponibles
Como compilar y ejecutar
El proyecto fue desarrollado en Visual Studio usando una aplicacion de consola en C++. Para abrirlo
y ejecutarlo:
1. Abrir Visual Studio.
2. Crear un nuevo proyecto: Archivo > Nuevo > Proyecto.
3. Seleccionar Aplicacion de consola y elegir C++ como lenguaje.
4. Copiar el contenido de segundo_parcial.cpp en el archivo principal del proyecto.
5. Presionar F5 o el boton Iniciar para compilar y ejecutar.
Como usarlo
Al abrir el programa aparece un menu con 7 opciones. Primero hay que registrar los estudiantes
(opcion 1) y luego se pueden consultar las demas opciones. Si se intenta ver datos sin haber
registrado antes, el programa lo indica.
